# 內訓與客服 QA 對照表

本表對照說明中心條目與 **內訓教材**、**客服 QA 題庫** 使用方式，方便新人訓練與客服快速查找答案。答案皆以說明中心為單一來源，避免多版本不一致。

## 使用方式

- **內訓**：依「新人必讀」「進階功能」分組，直接連結對應說明中心頁面作為教材。
- **客服**：遇到用戶問題時，依關鍵字或分類找到對應條目，可複製說明中心連結給用戶或依內容口頭/文字回覆。

## 內訓分組建議
