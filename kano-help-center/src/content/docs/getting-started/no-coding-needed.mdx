## 問題

不確定是否要有技術背景才能建站與維護網站。

## 解答

**完全不需要。** Kano 針對餐飲與微型創業設計，您只需像填寫問卷一樣輸入資訊，Kano 會自動為您生成專業的一頁式網頁。

- 建站：依精靈步驟填寫品牌名稱、服務內容、視覺與按鈕即可。
- 編輯：在編輯器中修改文字、圖片與模組，無需寫程式。
- 發布：一鍵發布，系統會處理網域與 SSL。

若有操作上的疑問，可參考說明中心或使用後台右下角的 Kano Agent 即時詢問。
