## 問題

想確認 Kano 網站是否安全、是否有 SSL 憑證，顧客填寫資料是否受保護。

## 解答

**安全是我們的標配。** 所有 Kano 網站（包含免費版）皆 **內建 SSL 安全加密憑證**，網址開頭為 **https**，確保顧客在瀏覽或填寫資訊時受到保護。

- 您無需自行申請或安裝 SSL。
- 發布後系統會自動處理憑證與加密連線。
- 適用 Kano 子網域與 Pro 自訂網域。
