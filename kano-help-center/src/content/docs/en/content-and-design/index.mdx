## Modules

Add, edit, and reorder modules (cover, image+text, cards, location, etc.) in the editor with live preview.

## AI

- Check that **AI credits** are sufficient
- Watch for **rate limits**

## Images

- **JPG, PNG, WebP**; max **10MB** per file
- Avoid more than **5 uploads per minute**

## Brand colors

Use **#** followed by six hex digits (e.g. `#FF5733`). Set both light and dark mode colors.
