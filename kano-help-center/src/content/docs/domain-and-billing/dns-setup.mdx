## 概述

Pro 版支援自訂網域。綁定流程為：在 Kano 後台輸入欲使用的網域，系統會顯示需要的 DNS 紀錄；您再到網域註冊商（如 Godaddy、Cloudflare）後台新增或修改對應紀錄即可。

## 設定步驟

### 1. 在 Kano 取得 DNS 指示

- 進入網站後台 **[設定]**，找到 **網域** 或 **自訂網域** 區塊。
- 輸入您要綁定的網域（例如 `www.yoursite.com` 或 `yoursite.com`）。
- Kano 會顯示需要設定的 **CNAME** 或 **A 紀錄** 與對應目標值，請照畫面指示操作。

### 2. 到網域註冊商設定

- 登入您購買網域的平台（如 Godaddy、Cloudflare、台灣網路資訊中心等）。
- 找到該網域的 **DNS 設定** / **DNS 管理** / **名稱伺服器**。
- **新增或編輯紀錄**：
  - 若 Kano 要求 **CNAME**：主機填 `www`（或畫面指定），指向 Kano 提供的目標網址。
  - 若 Kano 要求 **A 紀錄**：主機填 `@` 或 `www`（依畫面），IP 填 Kano 提供的位址。
- 儲存後等待 **DNS 傳播**，通常數分鐘至 48 小時內生效。

### 3. 回 Kano 驗證

- DNS 生效後，回到 Kano [設定] 的網域區塊，依畫面提示 **驗證** 或 **完成綁定**。
- 綁定成功後，您的網站即可透過自訂網域與 https 存取。

## 常見問題

- **一直顯示未驗證？** 請確認 DNS 紀錄與 Kano 顯示的完全一致（主機名稱、類型、目標值），並等待傳播時間。
- **只綁定 www 或只綁定根網域？** 依 Kano 畫面指示設定；若需同時支援 `yoursite.com` 與 `www.yoursite.com`，兩筆紀錄可能都需設定。
