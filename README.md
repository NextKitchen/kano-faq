# 🛠️ Kano 常見問題 (FAQ)

## 📋 帳號登入與系統操作
* **登入失敗：** 若出現「驗證已過期」，請確認網址為 `kano.site` 並從登入頁重新點擊「用 Google 登入」操作一次。
* **自動儲存：** 每 2 秒自動儲存一次，但草稿僅保留 14 天，超過期限系統將自動清理。
* **按鈕無法點擊：** 請檢查 Step 1-4 的必填項（品牌名稱、服務圖片、品牌標誌、顏色、自訂按鈕網址）是否填寫完整。

## 🔍 SEO 與搜尋排名
* **如何排在 Google 第一頁？**
    * 在「品牌標語」加入 `[地區] + [品牌類型]`，例如「中山區最濃郁的熟成咖哩」。
    * 提交 Sitemap 至 Google Search Console 進行抓取。
    * 使用 Kano 內建的第三方數據整合與 AI 優化助手來佈局關鍵字。

## 📈 流量與行銷追蹤
* **FB Pixel：** 到 [設定] > [追蹤] 貼入 Pixel ID，Kano 會自動紀錄誰點擊了訂位按鈕。這能幫助您投放精準的再行銷廣告。
* **GTM：** 貼入 GTM ID 後，不需更動代碼即可自由擴充 LINE 轉換代碼或 Google 廣告追蹤。

## 🌐 網域與方案
* **網域購買：** 可在 Godaddy、Namecheap 或 Cloudflare 等平台購買，並於後台修改 DNS 即可對接。
* **Kano 優勢：** 即使沒有網域也能用 Kano 網址一鍵發布，生意先開張，專業網域隨後補上。
* **方案差異：** Pro 版支援自訂網域、移除浮水印、提供進階 SEO 分享與數據分析功能。
