# 🛠️ Kano 常見問題 (FAQ) 

> **Next Kitchen 增長策略建議：** 建立網站只是第一步，善用下方的 SEO 與行銷追蹤工具，才能讓網站為您的品牌帶來實質營收成長。

## 📋 帳號登入與系統操作
* **登入時出現「驗證已過期」怎麼辦？**
    請關閉多餘分頁、確認網址為 `kano.site`，並從登入頁重新點擊「用 Google 登入」操作一次 [cite: 334]。
* **進度會消失嗎？**
    系統每 2 秒自動儲存一次。但請注意，草稿僅保留 14 天，超過期限系統將自動清理 [cite: 336, 337]。
* **為什麼「下一步」按鈕無法點擊？**
    通常是因為必填項未填完。請檢查品牌名稱、服務圖片、品牌色（深/淺皆須設定）以及自訂按鈕的網址格式是否正確 [cite: 339, 340, 341, 342]。

## 🎨 網站內容與設計
* **AI 功能無法使用？**
    請檢查帳號內的 AI 點數是否充足，或是否達到該功能的冷卻頻率限制 [cite: 345, 346]。
* **圖片上傳規範：**
    支援 JPG、PNG、WebP，單檔最大 10MB。若上傳過於頻繁（每分鐘超過 5 次），請稍後再試 [cite: 349, 350, 351]。
* **品牌色設定：**
    必須使用 # 開頭的六位數色碼（如 #FF5733） [cite: 347]。

## 🔍 SEO 與搜尋排名（讓客人主動找到你）
* **如何排在 Google 第一頁？**
    * [cite_start]**關鍵字公式：** 在標語加入 `[地區] + [品牌類型]`，例如「中山區最濃郁的熟成咖哩」 [cite: 381]。
    * [cite_start]**圖片描述：** 上傳圖片時填入名稱，增加 Google 圖片搜尋曝光 [cite: 382]。
    * [cite_start]**主動出擊：** 將網址提交給 Google Search Console (Sitemap) [cite: 389]。
* **為什麼點擊率高卻沒下單？**
    [cite_start]建議優化按鈕文案。將「聯絡我們」改為「領取今日隱藏優惠」，並使用深色背景按鈕以增加對比度 [cite: 385, 386]。

## 📈 流量與行銷追蹤（FB Pixel & GTM）
* **FB Pixel (數位監視器)：**
    紀錄訪客行為並投放「再行銷廣告」。只需將 Pixel ID 貼進 Kano [設定] > [追蹤] 即可，免寫代碼 [cite: 393, 394]。
* **GTM (代碼管理工具)：**
    在裡面自由裝載 Google 廣告追蹤或 LINE 轉換代碼。將 GTM ID 貼入 Kano 後，所有設定將自動生效 [cite: 395]。

## 🌐 網域與訂閱方案
* [cite_start]**網域購買：** 推薦去 Godaddy 或 Cloudflare 購買。即使還沒買網域，也能先用 Kano 網址一鍵發布，生意先開張 [cite: 398, 402]。
* [cite_start]**Pro 版價值：** 支援自訂網域、可移除浮水印，並提供進階 SEO 分享與數據分析工具 [cite: 367]。
* [cite_start]**取消訂閱：** 網站會持續運作至「訂閱到期日」為止，之後才會降回免費版 [cite: 371]。
