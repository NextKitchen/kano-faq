## Login issues

If you see "Verification expired":

1. Close the tab
2. Make sure the URL is `kano.site`
3. Click "Sign in with Google" again on the login page

## Auto-save

- Saves every **2 seconds**
- Drafts are kept for **14 days** only

## Buttons not clickable

Check that all required fields are filled, e.g. brand name, service images, brand colors, button URL format.
